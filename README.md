Ruang Ekspresi

Proyek ini merupakan hasil slicing desain menggunakan React, Vite, dan Tailwind CSS. Tampilan dibuat berdasarkan desain Figma dan disusun menjadi beberapa komponen agar struktur kode lebih teratur dan mudah dipelihara.

Fitur Utama
- Navigasi dengan menu responsif
- Hero section
- Empat pilar program
- Jadwal kegiatan
- Showcase project mahasiswa dengan navigasi
- Bagian testimoni
- Footer sederhana

Teknologi yang digunakan
- React
- Vite
- Tailwind CSS
- React Icons

Cara Menjalankan Proyek
1. Pastikan Node.js telah terpasang.
2. Jalankan perintah berikut:

npm install
npm run dev

3. Aplikasi akan berjalan pada alamat yang ditampilkan di terminal.

Struktur Folder Singkat
src/
├─ components/
├─ pages/
├─ assets/
├─ index.css
└─ main.jsx


Deployment
Proyek ini dapat dijalankan menggunakan GitHub Pages, Netlify, atau layanan hosting statis lainnya.

Lisensi
Proyek ini dibuat untuk keperluan pembelajaran.
